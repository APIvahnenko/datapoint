About this site

Purpose

This site is a way for me to showcase my work and it gives me a reason to improve my skills and understanding. I'm a visual person, who still learning a lot about this world and finds it interesting. I hope you found something interesting here for you.

Development

This site was written using HTML, CSS and JavaScript. It uses a custom Rust static site generator and is hosted on GitHub Pages. On each push a GitHub Actions Workflow builds the site using Zola and deploys the resulting files to a GitHub Page served using a domain name provided by Amazon Web Services.

Performance testing was conducted in the latest versions of Chrome, Firefox, and Safari (mobile and desktop) browsers.

The fonts used are Halvatica and Serif.

This site is using the Axiom for request logs and server-side analytics.

A sitemap is available and there is an RSS feed for new posts.

Accessibility

This website is regularly checked for performance and accessibility issues using Google Lighthouse. This site follows Web Content Accessibility Guidelines (WCAG) 2.2 to AA standard. Screen reader ease of use is checked using Voice Over and audited using WAVE.

If you discover an accessibility problem or any other issue, please open a git issue or send me an email, I'll try to fix any issues as soon possible.
